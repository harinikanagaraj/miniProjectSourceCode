# Bank Management System in C

A simple Bank Management System developed using the C programming language and Random Access File handling.

## Features

- Add new bank account
- Update account balance
- Deposit money
- Withdraw money
- Delete account
- Search account
- Store records in text file
- Password protection
- Random access file handling

---

## Technologies Used

- C Programming
- File Handling
- Structures
- Random Access Files
- GCC Compiler

---

## Project Files

| File Name | Description |
|-----------|-------------|
| trans.c | Main source code |
| credit.dat | Binary file for account storage |
| accounts.txt | Text file containing account details |
| README.md | Project documentation |
